rvm-prompt

To display the current ruby selection string in your prompt there are two steps necessary.

First copy scripts/rvm-prompt to somewhere a bin directory in your path and then add this to your profile file:

PS1="$PS1\$(rvm-prompt) "

This ends up looking like this during usage:

∴ rvm 1.9.1 [ruby:1.9.1:p243] ∴ [ruby:1.9.1:p243] ∴ rvm 1.8.6 [ruby:1.8.6:p383] ∴ rvm jruby [jruby:1.3.1] ∴
