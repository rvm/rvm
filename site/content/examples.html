Examples of using rvm

RVM version

∴ rvm -v rvm 0.0.19 (2009.08.25) [http://github.com/wayneeseguin/rvm]

Available rvm Ruby Interpreter installations

∴ rvm list ruby: - ruby-1.8.6-p369 - ruby-1.8.7-p160 - ruby-1.8.7-p174 - ruby-1.9.1-p243 jruby: - jruby-1.3.1 ree: default: - system (ruby 1.8.6 (2008-08-11 patchlevel 287) [universal-darwin9.0])

∴ rvm info ruby: interpreter: "ruby" version: "1.9.1p243" date: "2009-07-16" platform: "i386-darwin9.8.0" patchlevel: "2009-07-16 revision 24175" full_version: "ruby 1.9.1p243 (2009-07-16 revision 24175) [i386-darwin9.8.0]" homes: gem: "/Users/wayne/.gem/ruby/1.9.1" ruby: "/Users/wayne/.rvm/ruby-1.9.1-p243" binaries: ruby: "/Users/wayne/.rvm/ruby-1.9.1-p243/bin/ruby" irb: "/Users/wayne/.rvm/ruby-1.9.1-p243/bin/irb" gem: "/Users/wayne/.rvm/ruby-1.9.1-p243/bin/gem"

Install jRuby (default version is 1.3.1)

∴ rvm install jruby

Use Ruby 1.9.1, installs if necessary

∴ rvm use ruby -v 1.9.1

Equivalent to "rvm use ruby -v 1.9.1", due to defaults

∴ rvm 1.9.1

Use Ruby 1.8.6, installs if necessary

∴ rvm 1.8.6

Switch to gems directory for current ruby

∴ rvm gemdir /Users/wayne/.gem/ruby/1.9.1

Switch to the system gems directory

∴ rvm gemdir system /Library/Ruby/Gems/1.8

Switch to the system user gems directory

∴ rvm gemdir system user /Users/wayne/.gem/ruby/1.8

Switch to gems directory for ruby jruby (1.3.1)

∴ rvm gemdir jruby /Users/wayne/.gem/jruby/1.8

Use the system default (as if no rvm)

∴ rvm use default

Reset to pre-rvm state.

∴ rvm reset

Uninstall rvm installed 1.8.7 version

∴ rvm uninstall 1.8.7

Install gems from ~/.gem/ruby/1.8/

∴ rvm gemdup default

Install gems from system gem dir

∴ rvm gemdup system
