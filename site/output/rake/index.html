JSON Summary

Along with 'rvm rake X' you can add the --json flag prior to the word 'rake' and a JSON summary will be printed out at the end of the run.

YAML Summary

Along with 'rvm rake X' you can add the --yaml flag prior to the word 'rake' and a YAML summary will be printed out at the end of the run.
