Operating System pre-requirements

Debian/Ubuntu

For jRuby (if you wish to use it) you will need:

sudo apt-get install sun-java6-bin sun-java6-jre sun-java6-jdk

For ree (if you wish to use it) you will need:

sudo apt-get install libreadline5-dev libssl-dev bison

Gentoo

For jRuby (if you wish to use it) you will need:

sudo emerge dev-java/sun-jdk dev-java/sun-jre-bin

Arch Linux

For jRuby (if you wish to use it) you will need:

sudo pacman -Sy jdk jre

RH/Centos/FC

For ree (if you wish to use it) you will need:

yum install -y rpm-build gcc gcc-c++ redhat-rpm-config ; then download and rpmbuild and install the sdk, Have fun...

Mac OS X

Be sure that you have XCode Tools installed in order to use rvm.

If you intend on installing MacRuby you must install LLVM first.
