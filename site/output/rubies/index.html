Getting Head

Getting the head/trunk version of rubies is very simple. For any interpreter which has head support simply prefix -head after the interpreter. 

∴ rvm install ree-head
∴ rvm use ree-head

Note that ase of this writing 1.9.2 has no repository branch and thus "1.9.1-head" IS "1.9.2-head".
